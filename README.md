# WorkShopSearcher
勉強会検索iOSアプリ開発用リポジトリ

## 使用技術
- [RxSwift](https://github.com/ReactiveX/RxSwift)
- [Quick](https://github.com/Quick/Quick)
- [Nimble](https://github.com/Quick/Nimble)

## テスト
[Quick](https://github.com/Quick/Quick)を使ってテストをしていきます。

## 環境
- OS version <= macOS X Mojave 10.14.3 
- Swift5
- Carthage
